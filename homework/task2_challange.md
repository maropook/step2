key も value も 20文字以下（定数時間で処理可能）
key に重複はない（同一の名前のユーザは受け付けない）

put("alice", "1111") のあとで、put("alice", "1234") が呼ばれたら、"alice" の value が "1234" にアップデートされます。（いわゆる Python の辞書だと思ってください！）

keyもvalueもalphabetと空白だと仮定すると、27**20種類のkeyが存在する。27 **20の配列を準備する。a = 0, b = 1,,, " "=27とし、27進数で表してから10進数に変換し、違うkeyでは重複しないようなindexが作成でき、O(1)で検索・追加・削除できる。
